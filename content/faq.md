# Veelgestelde Vragen - Particulieren

{{< details title="Hoe lever ik een 3D-model aan?" closed="true" >}}
Je kunt je STL- of STEP-bestand direct uploaden via het contactformulier of doorsturen per mail. Heb je geen eigen model? Geef dan gerust een woordje uitleg over je idee, dan kijken we samen naar de mogelijkheden.
{{< /details >}}

{{< details title="Welke materialen gebruiken jullie?" closed="true" >}}
Wij printen exclusief met TM3D filamenten van topkwaliteit. Dit garandeert constante kleurechtheid, maatnauwkeurigheid en sterke printresultaten. Momenteel werken we nog een kleine stock aan e-sun filamenten op.
{{< /details >}}

{{< details title="Hoe wordt de prijs berekend?" closed="true" >}}
Onze particuliere tarieven zijn heel helder: een vaste starttarief van €12,00 + 2 × de werkelijke materiaalkosten.
{{< /details >}}

{{< details title="Wat is de levertijd?" closed="true" >}}
Gemiddeld wordt een print binnen de 4 dagen (na betaling) geproduceerd en verzonden of klaargezet voor afhaling in Edegem.
{{< /details >}}

{{< details title="Kan ik mijn bestelling ook afhalen?" closed="true" >}}
Jazeker! Afhalen in Edegem is mogelijk en kost niets extra. Anders wordt het zorgvuldig verpakt en verzonden. De verzendkosten zijn volledig voor de klant.
{{< /details >}}

# Veelgestelde Vragen - Bedrijven (B2B)

{{< details title="Kan ik bij jullie terecht voor kleine productseries?" closed="true" >}}
Ja, van enkele functionele prototypes tot kleine series ben je bij ons aan het juiste adres voor flexibele en snelle productie.
{{< /details >}}

{{< details title="Hoe is de zakelijke prijsopbouw geregeld?" closed="true" >}}
Voor B2B hanteren we onze basisformule (€12,00 + 2 × filamentkost). Eventuele extra uren voor CAD-voorbereiding, technische optimalisatie of intensieve nabewerking worden gerekend aan een helder uurtarief van €18,00.
{{< /details >}}

{{< details title="Ontvang ik altijd een offerte vooraf?" closed="true" >}}
Absoluut. Elk zakelijk bestand wordt eerst gecontroleerd op printbaarheid en toleranties, volledig gratis. Je onvangt altijd een vrijblijvend en gedetailleerd prijsvoorstel voordat we starten.
{{< /details >}}

{{< details title="Welke bestandsformaten worden ondersteund?" closed="true" >}}
Naast standaard STL-bestanden accepteren we voor technische en functionele B2B-onderdelen ook STEP-bestanden voor maximale nauwkeurigheid.
{{< /details >}}

{{< details title="Krijg ik professioneel advies over materiaalkeuze?" closed="true" >}}
Zeker. Omdat we uitsluitend met hoogwaardige filamenten werken, denken we graag mee over het juiste type kunststof. (bijv. op het gebied van sterkte, temperatuurbestendigheid of mechanische belasting).
{{< /details >}}
